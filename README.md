# Dotfiles

[![Twitter](https://img.shields.io/twitter/url/https/github.com/lukeify/productivity.svg?style=social)](https://twitter.com/intent/tweet?text=Wow:&url=https%3A%2F%2Fgithub.com%2Flukeify%2Fproductivity)

My dotfiles for provisioning a new Mac. 


## Getting Started

Generally, manual installation of these files is not needed, as these are installed automatically from [lukeify/macos-provision](https://github.com/lukeify/macos-provision), this repository merely acts as the source.

### Installation

Dotfiles can be installed manually if desired, or otherwise for alterations:

```
git clone https://github.com/lukeify/dotfiles
cd dotfiles
```

## Built With

* Bash
* Text editor of your choice

## Versioning

This repository will be versioned datewise as necessary, and the latest release will be downloaded when running my macos-provisioning script.
